JavaScript Npm "js-yaml" Demo
=============================

使用"js-yaml"库对YAML文件进行转换。

```
npm install
npm run demo
```

输出：

```
---------- parse yaml file ------------
{"name":"YAML & JSON","keywords":["JavaScript","YamlJS",{"formats":["YAML","JSON"]}]}
---------- dump object to YAML text ----------
name: YAML & JSON
keywords:
  - JavaScript
  - YamlJS
  - formats:
      - YAML
      - JSON
```
